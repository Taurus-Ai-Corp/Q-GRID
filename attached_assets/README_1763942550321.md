## Quick Start

### 1. Install Dependencies

```bash
cd quantum-crypto-core
npm install

cd ../hedera-integration
npm install
```

### 2. Set Up Hedera Testnet

```bash
# Get testnet account from https://portal.hedera.com
export HEDERA_ACCOUNT_ID="0.0.123456"
export HEDERA_PRIVATE_KEY="302e..."
```

### 3. Run Examples

```bash
# Quantum crypto example
node quantum-crypto-core/examples/mldsa-example.js

# Hedera integration example
node hedera-integration/examples/hedera-quantum-example.js
```
